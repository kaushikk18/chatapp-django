# chatapp-django

## DESCRIPTION
A real time chatapp created using django 
### Procedure
### Simply clone this repo
    $ git clone [repo link]
    $ cd project_name
    $ pipenv shell
    $ pipenv install
    $ python manage.py runserver

change the python_version to your system version in your Pipfile

## Technology Used
* Backend 
    * Django/Python
    * Sqlite3
* Frontend
    * HTML
    * CSS
    * Javascript
